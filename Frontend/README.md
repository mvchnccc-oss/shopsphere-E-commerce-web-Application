# 🛒 ShopMart E-commerce Web Application

A modern, responsive e-commerce interface built with **Next.js** and **Tailwind CSS**.

## ✨ Features
- 📱 Fully Responsive Design (Mobile & Desktop).
- 🛍️ Dynamic Product Fetching using External API.
- 🛒 Interactive Shopping Cart and UI Components.

## 🚀 Tech Stack
- **Framework:** Next.js
- **Styling:** Tailwind CSS & shadcn/ui
- **Language:** TypeScript / JavaScript
